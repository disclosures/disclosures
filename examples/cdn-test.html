What is Disclosures?

Disclosures is a lightweight library for animated disclosure components with smooth height transitions.

Plus/Minus Indicator

How does it work?

It uses the Web Animations API to smoothly animate the height of details elements when they open and close.

FAQ Variant (Exclusive Group)

Is it accessible?

Yes! It uses native details/summary elements, respects prefers-reduced-motion, and is fully keyboard navigable.

What browsers are supported?

All modern browsers that support the Web Animations API (Chrome, Firefox, Safari, Edge).

Does it have dependencies?

Zero dependencies! It's pure TypeScript/JavaScript with CSS.
